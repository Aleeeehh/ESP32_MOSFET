# MOSFET

Un semplice progetto PlatformIO su ESP32, che aumenterà di complessità col tempo.

## Hardware Richiesto

- ESP32DevKit
- ESP3-CAM
## Installazione

1. Clona questa repository
2. Apri il progetto in PlatformIO
3. Compila e carica il codice sull'esp32

## Compilazione

```bash
pio run
```

## Upload

```bash
pio run --target upload
```

## Monitor Seriale

```bash
pio device monitor
```
